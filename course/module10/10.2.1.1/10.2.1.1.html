Well-Known Application Layer Protocols and Services

Common Application Layer Protocols

There are dozens of application layer protocols, but on a typical day you probably use only five or six. Three application layer protocols that are involved in everyday work or play are:

  • Hypertext Transfer Protocol (HTTP)

  • Simple Mail Transfer Protocol (SMTP)

  • Post Office Protocol (POP)

These application layer protocols make it possible to browse the web and send and receive email. HTTP is used to enable users to connect to web sites across the Internet. SMTP is used to enable users to send email. And POP is used to enable users to receive email.
