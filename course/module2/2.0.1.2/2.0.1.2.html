It Is Just an Operating System!

In this activity, imagine that you are employed as an engineer for a car manufacturing company. The company is currently working on a new car model. This model will have selected functions which can be controlled by the driver giving specific voice commands.

Design a set of commands used by this voice-activated control system, and to identify how they are going to be executed. The functions of the car that can be controlled by voice commands are:

  • Lights

  • Wipers

  • Radio

  • Telephone set

  • Air conditioning

  • Ignition
